## 中文簡介

定量資料分析互動式教學。14 章 / 3 區塊串連 qPCR / Western Blot / ELISA / Flow Cytometry → EDA → 假設檢定 / FDR / Effect size → 組學 → AI/ML 報告規範。

## English Description



Bilingual (zh-Hant / English) interactive tutorial covering 14 topics across:
- Experimental quantification (qPCR, Western blot, ELISA, flow cytometry, image analysis)
- Statistical inference (EDA, hypothesis testing, multiple-testing correction, regression, effect size)
- Advanced topics (omics quantification, visualization, reproducibility, AI/ML)

Open `index.html` in any modern browser. Click the **EN / 中文** toggle in the top-right to switch language. Includes interactive sliders, Chart.js simulations, and R + Python dual code blocks.

Related sub-sites:
- `references/index.html` — references with DOI links and reporting standards
- `qda-quiz/index.html` — interactive quiz across all 14 topics
